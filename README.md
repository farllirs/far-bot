## English

### What's New in v2.0.0

- **Slash Commands Support**: Full support for Discord slash commands (/)
- **Bot Avatar & Server Display**: See your bot's avatar and connected servers
- **Improved Command Editor**: Better undo/redo, mobile support, more templates
- **Documentation Links**: Quick access to docs from the panel
- **Enhanced UI**: Better mobile experience and visual improvements
- **Bug Fixes**: Resolved command execution issues

### Quick Start

\`\`\`bash
# Clone and install
git clone https://github.com/yourusername/far-bot.git
cd far-bot
pip install -r requirements.txt

# Launch
python launcher.py
\`\`\`
